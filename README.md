# Disease Surveillance India

This project focuses on *exploratory data analysis (EDA)* of disease outbreak data to uncover **patterns, trends, and anomalies** across regions and time.

---

## Overview

The dashboard is built using **Streamlit** and enables interactive exploration of disease trends across **time, geography, and categories**.

It is designed to support both:
- **Descriptive analysis**
- **Deeper investigation of unusual observations** within the dataset

---

## Dashboard Structure

The application is organized into the following sections:

### Overview
*Summary metrics and high-level distributions of cases*

### Trends
*Temporal analysis across years and months*

### Geographic Analysis
*State-level comparisons and reporting patterns*

### Disease Analysis
*Focused exploration of individual diseases*

### Seasonality
*Monthly patterns and recurring trends*

### Anomaly Detection
*Identification and investigation of unexpected mortality events*

---

## Technology Stack

- **Python**
- **Streamlit**
- **Pandas**
- **Plotly**

---

## Notes on Data Interpretation

The dataset represents *reported surveillance data* and should not be interpreted as a direct measure of actual disease burden.

Variations may reflect:
- Differences in reporting infrastructure  
- Diagnostic accessibility  
- Data entry practices  

The insights generated by this dashboard are intended to **guide further analysis**, not serve as definitive conclusions.
